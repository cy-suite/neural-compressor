Base class for patched modules.

The patched module is used to replace the original float module in the model at prepare and convert stages. To make the patched module compatible with the module-agnostic patching process, the patched module should provide a set of module configurations by implementing the following abstract methods:

  • get_type

  • get_module_type

The patched module should also implement the following methods for its specific functionalities:

  • forward_measure

  • forward_quant

  • forward_qdq(Optional), only required when using Q-DQ mode for quantization stage.
